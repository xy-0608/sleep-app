# 🌙 Sleep App

一款美观的睡眠助眠应用，提供白噪音、冥想音乐、睡眠追踪功能。

## 功能特性

- 🔊 **多种白噪音** - 雨声、篝火、海浪、森林、风扇等
- 🧘 **冥想音乐** - 舒缓的冥想音乐帮助放松
- 📊 **睡眠追踪** - 记录你的睡眠时长
- ⏰ **智能闹钟** - 温柔唤醒
- 🎨 **美观界面** - 深色主题，舒适体验

## 技术栈

- Flutter 3.x
- Dart 3.x
- Provider 状态管理
- Just Audio 音频播放

## 项目结构

```
lib/
├── main.dart           # 入口文件
├── models/             # 数据模型
├── providers/          # 状态管理
├── screens/            # 页面
├── widgets/            # 组件
└── utils/              # 工具
```

## 快速开始

1. 安装依赖
```bash
flutter pub get
```

2. 运行项目
```bash
flutter run
```
